Expand description

The set of ASCII values that are non-printable characters and require special handling by a terminal emulator.

Variants

Bell

(BEL) Plays a terminal bell or beep. Ctrl + G, or '\a'.

Backspace

(BS) Backspaces over the previous character before (to the left of) the cursor. Ctrl + H, or '\b'.

Tab

(HT) Inserts a horizontal tab. Ctrl + I, or '\t'.

NewLine

(LF) Moves the cursor to the next line, i.e., Line feed. Ctrl + J, or '\n'.

VerticalTab

(VT) Inserts a vertical tab. Ctrl + K, or '\v'.

NewPage

(FF) Inserts a page break (form feed) to move the cursor/prompt to the beginning of a new page (screen). Ctrl + K, or '\v'.

CarriageReturn

(CR) Moves the cursor to the beginning of the line, i.e., carriage return. Ctrl + M, or '\r'.

Escape

(ESC) The escape character. ESC, or '\e'.

Delete

(DEL) Deletes the next character after (to the right of) the cursor. DEL.
