ApicRegisters

A structure that offers access to APIC/xAPIC through its I/O registers. Definitions are based on [Intel's x86 Manual Vol 3a, Table 10-1] (https://www.intel.com/content/dam/www/public/us/en/documents/manuals/64-ia-32-architectures-software-developer-vol-3a-part-1-manual.pdf).

LocalApic

This structure represents a single APIC in the system, there is one per core.

INTERRUPT_CHIP

The interrupt chip that is currently configured on this machine. The default is InterruptChip::PIC, but the typical case is APIC or X2APIC, which will be set once those chips have been initialized.
