A structure that offers access to APIC/xAPIC through its I/O registers. Definitions are based on [Intel's x86 Manual Vol 3a, Table 10-1] (https://www.intel.com/content/dam/www/public/us/en/documents/manuals/64-ia-32-architectures-software-developer-vol-3a-part-1-manual.pdf).

Fields

lapic_id: Volatile<u32>

This Lapic's ID. Some systems allow setting the ID, but it is typically read-only. Only the top 8 bits are relevant, so bit shift it to the right by 24 bits to get the actual ID.
